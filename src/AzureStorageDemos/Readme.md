
### To start the Azure Storage Emulator:

Open Developer Command Prompt in **Administrator** mode

```
> azurite --skipApiVersionCheck
```

----

Open **Azure Storage Explorer** to view the Azure Storage Emulator services for:
- Blob Storage
- Table Storage
- Queue Storage

----

term       | concept | meaning
-----------|---------|-------------
expandable | volume  | able to be made larger or more extensive	
expendable | utility | designed to be used only once and then abandoned or destroyed
extendable | length  | able to be made longer or larger.


| Aspect      | Scale-Up                                          | Scale-Out                                               |
| :---------- | :------------------------------------------------ | :------------------------------------------------------ |
| approach    | Increase capacity of a single machine             | Add more machines                                       |
| performance | Boost performance of a single machine             | Distribute workload for concurrent processing           |
| capacity    | Constrained by max capacity of a single machine   | Virtually unlimited                                     |
| cost        | Higher upfront cost for high-performance hardware | Lower initial cost, distributed across multiple servers |
| resilience  | Single point of failure                           | Higher resiliency and redundancy                        |
